pub fn insertion_sort<T: Ord + Clone>(array_orig: ~[T]) -> ~[T]

Insertion sort takes an array of type T which has the Ord and Clone traits. Returns an array of type T that has been sorted. It has two loops, one nested in the other. The outer loop sets an immutable variable, val, to the value of the ith index and sets a mutable variable j to the value of i - 1. The inner loop executes while j +1 != 0 the length of the array and while the value of the jth element is greater than the value of val, and sets the value of the element to the "right" of j to the value of the jth element. j is then decremented. When j is equal to zero and decrments, it will fulfill the first termination condition of the inner while loop. After the termination of the inner loop, the element to the right of the jth element is assigned the value of val, and i increments.
