pub fn rubin_karp(array: &[~str], key: &str) -> (bool, uint, uint)

Takes an array of strings and successively iterates through each string. On each iteration, it compares the hash of a substring of equal length to the key to the hash of the key, and terminates either when a match is found or if all possibilities are exhausted. Returns a boolean to indicate if it was found, the index (uint) of the array holding the string in which it was found, and the index (uint) of that string at which the matching substring begins.
